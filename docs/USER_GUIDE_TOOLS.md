## Tools Overview

The Tools section provides quick utilities for property evaluation and analysis.

- Mortgage Calculator: enter price, down payment %, interest %, and years to get monthly payment.
- Compare Properties: input comma-separated property IDs to see summary stats.
- Price Analysis: provide a query (e.g., city/type) to get averages and medians.
- Location Analysis: enter a property ID to view city, neighborhood, lat/lon.
- Valuation (CE stub): estimate value from area and price per sqm.
- Legal Check (CE stub): basic risk score from pasted contract text.
- Data Enrichment (CE stub): address lookup (may require enabling flag).
- CRM Sync (CE stub): sync a contact if webhook is configured.

## How to Use
1. Open Tools page from the main navigation.
2. Fill in the form and click the action button.
3. Review results or error messages and adjust inputs if needed.
