# Documentation (Trae Rules)

## Code Docs
- **Docstrings**: Google style for public API. Document args, returns, raises.
- **Comments**: English. Explain "why". Use `TODO(#id):`.

## Project Docs
- **Structure**:
  - `docs/PRD.MD`: Requirements.
  - `docs/TESTING_GUIDE.md`: How to test.
  - `docs/DEPLOYMENT.md`: Build/deploy.
  - `docs/ARCHITECTURE.md`: System overview.
  - `docs/SECURITY.md`: Secrets & data.
- **Release**: `docs/RELEASE_NOTES.md`.
